# 📱 微信公众号文章导入操作指南

> **目的**:把 3 篇微信公众号最新文章手动嵌入到 foodvio 官网首页"最新动态" section
> **频率**:每次公众号发布新文章后做一次
> **位置**:index.html

---

## 🎯 为什么不能自动同步?

微信公众号**没有公开的 API** 给第三方网站读取文章列表。可行方案对比:

| 方案 | 难度 | 维护成本 | 稳定性 |
|---|---|---|---|
| **A. 手动替换 3 张卡的 HTML**(本指南方案)| ★ 极简单 | 每次发文章手动改 | ★★★★★ 100% |
| B. 用新榜/西瓜数据等第三方付费抓取 | ★★ 中等 | 需购买 SaaS + 配置 | ★★★ |
| C. 部署 Python 爬虫定期抓取 | ★★★★ 难 | 微信反爬频繁,维护累 | ★★ |

**我选 A 方案** — 最稳定 + 最快上手 + 0 成本。

---

## 📋 你需要做的 3 步(每次发文章后)

### Step 1 · 获取微信公众号文章 URL

发布文章后:
1. 在微信公众号后台 → 已发布文章 → 选文章 → 复制链接
2. URL 格式:`https://mp.weixin.qq.com/s/XXXXXXX`(s/ 后是文章唯一 ID)
3. 准备好以下信息:
   - 完整 URL
   - 文章标题(中文 + 英文)— 英文是你自己写的对应标题
   - 发布日期(格式 YYYY.MM,如 2026.05)
   - 分类标签(如"公众号文章" / "新品" / "行业洞察")

### Step 2 · 打开 `index.html`,搜索 `REPLACE_WITH_WECHAT`

会找到 3 处占位符,每处对应 1 张文章卡:

```html
<!-- 文章 1 - 待 Jerome 替换为微信公众号最新文章 -->
<a href="https://mp.weixin.qq.com/s/REPLACE_WITH_WECHAT_ARTICLE_URL_1" target="_blank" rel="noopener" class="news-card reveal delay-1">
  <div class="news-img" style="background: linear-gradient(135deg, #1B4332, #81B29A);">📰</div>
  <div class="p-7">
    <div class="font-display italic text-terra text-xs tracking-wider mb-3">YYYY.MM · 公众号文章</div>
    <h4 class="font-serif-sc font-medium text-forest text-lg leading-snug">
      <span data-zh>【待录入】微信公众号文章标题 1</span>
      <span data-en>[Placeholder] WeChat Article Title 1</span>
    </h4>
  </div>
</a>
```

### Step 3 · 替换 4 个位置(每张卡)

| 替换什么 | 改成什么 |
|---|---|
| `https://mp.weixin.qq.com/s/REPLACE_WITH_WECHAT_ARTICLE_URL_1` | **微信文章真实 URL** |
| `YYYY.MM · 公众号文章` | **2026.05 · 新品发布**(实际日期 + 分类)|
| `【待录入】微信公众号文章标题 1` | **微信文章真实中文标题** |
| `[Placeholder] WeChat Article Title 1` | **微信文章对应英文标题** |

3 张卡都改完(占位符 `_1` / `_2` / `_3`)→ 保存 → git push → 上线。

## 🖼️ 进阶 · 用真实封面图替代 emoji

如果想用微信文章的封面图(更专业):

### 第一种方法:用文章封面图 URL(简单)
1. 在公众号文章页右键封面图 → 复制图片地址
2. 在 HTML 里把:
   ```html
   <div class="news-img" style="background: linear-gradient(135deg, #1B4332, #81B29A);">📰</div>
   ```
   改成:
   ```html
   <div class="news-img" style="background-image: url('封面图URL'); background-size: cover; background-position: center;"></div>
   ```

⚠️ **注意**:微信图片有防盗链,在外部网站可能加载失败。如果失败,用第二种方法。

### 第二种方法:下载图片到本地(推荐)
1. 下载 3 张封面图到 `assets/news/` 目录:
   - `news-1.jpg` / `news-2.jpg` / `news-3.jpg`
2. 同时生成 WebP 版本(压缩):
   - `news-1.webp` / `news-2.webp` / `news-3.webp`
3. HTML 改为:
   ```html
   <div class="news-img" style="background-image: url('assets/news/news-1.jpg'); background-size: cover; background-position: center;"></div>
   ```
